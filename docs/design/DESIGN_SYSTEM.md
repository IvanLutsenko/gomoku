# Design System

## Принципы

1. **Тишина и пустота.** Большие поля воздуха. Контент группируется в плотные блоки, отделённые `>= 24 px`.
2. **Один акцент — золото.** Киноварь (`vermillion`) — второй тон системы (ревизия 2026-07, тон кнопок «Киноварь»): primary-кнопки, активные сегменты, тогглы-on, печати-ханко, статусные микроакценты (последний ход, запретная точка рэндзю). Никогда не используется как фон большого блока.
3. **Никаких градиентов как фоновых заливок.** Только тонкие радиальные на углах сцены ради лёгкой текстуры бумаги.
4. **Сетка.** Все отступы кратны 4 px. Шаг типографики — 12 / 13 / 14 / 16 / 22 / 28 / 36 / 40 / 56 px.
5. **Углы.** Ревизия 2026-07 (форма «Округлые»): кнопки и сегмент-контролы — `corner_radius = 10`, печати/ханко — `5–6`, тогглы — овал `12 px`. Большие радиусы по-прежнему запрещены.

## Типографика

См. `tokens/typography.json`. Два семейства:

- **Noto Serif** + **Noto Serif JP** — заголовки, числа партий и иероглифы (五目).
- **Inter** — UI-текст, кнопки, метки.

| Стиль | Семейство | Размер | Вес | Letter spacing |
|---|---|---|---|---|
| `display` | Noto Serif JP | 56 | 400 | -1 |
| `title` | Noto Serif | 36 | 400 | 0 |
| `subtitle` | Noto Serif | 22 | 400 | 0 |
| `body` | Inter | 14 | 400 | 0 |
| `body_strong` | Inter | 14 | 600 | 0 |
| `button` | Inter | 16 | 500 | 0.3 |
| `button_small` | Inter | 14 | 500 | 0.3 |
| `caption` | Inter | 13 | 400 | 0 |
| `caption_italic` | Noto Serif italic | 13 | 400 | 0 |
| `meta` | Inter | 11 | 500 | 1.5 (caps) |
| `label_caps` | Inter | 12 | 500 | 2 (caps) |

В Godot: загрузить `.ttf`, в `Theme` → `default_font` поставить Inter, для каждого Label/Button с заголовочным стилем — переопределить `theme_override_fonts/font` на Noto Serif. Letter spacing в Godot выставляется через `theme_override_constants/separation` для контейнеров; для текста — добавлять пробелы (как в `G O M O K U`) или использовать `BBCode`-разрядку в `RichTextLabel`.

## Spacing & размеры

| Токен | px | Где |
|---|---|---|
| `space_xs` | 4 | внутри тогглов, иконки |
| `space_sm` | 8 | между мелкими блоками |
| `space_md` | 14 | паддинг кнопки |
| `space_lg` | 24 | поля экрана |
| `space_xl` | 32 | заголовок ↔ блок |
| `space_2xl` | 40 | большие отрывы |
| `screen_padding_h` | 24 | левый и правый отступ экрана |

Кнопки (ревизия 2026-07 — тон «Киноварь», форма «Округлые»):
- **Primary**: `bg = vermillion`, `fg = #f7f1e4`, без бордера, `padding = 18 24` (большая) / `13 16` (компактная), `corner_radius = 10`.
- **Secondary**: `bg = transparent`, `fg = ink`, бордер `1 px line_firm`, `corner_radius = 10`.
- **Tertiary** (текст-кнопка «← Назад»): без бордера, `fg = muted`, `font_size = 14`.

Поля и сегменты:
- Сегмент-контрол — горизонтальная группа кнопок без зазоров, общая рамка `1 px line_firm`, `corner_radius = 10` (скругление только на внешних углах крайних сегментов). Активный — `bg = vermillion, fg = #f7f1e4`.

Доска:
- Cell size: 22 px, padding внутрь рамки доски: 14 px.
- Линии решётки: `0.6 px`, `ink @ 0.7` (light) / `0.45` (dark).
- Точки хоси (звёзды): радиус 1.8 px, на (3,3), (3,11), (7,7), (11,3), (11,11).
- Камни: радиус ≈ `cell × 0.42`. Чёрный на светлой — радиальный градиент `#4a4640 → #0d0c0a`. Чёрный на тёмной — осветлённый сланец `#9a8f80 → #1a1814` + обводка `0.5 px rgba(255,253,245,0.4)`, иначе не видимы. Белый — `#fff → #d8d2c4` + `0.5 px` обводка `rgba(0,0,0,0.18)`.

## Тон копирайта

- Краткий, без восклицаний.
- Никогда не «Поехали!», «Готовы?» и подобного. Нейтральные глаголы: «Начать», «Продолжить», «В меню».
- Подзаголовки — фразами, без точки. Пример: «Пять камней в ряд».
